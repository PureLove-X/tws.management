# ============================================================
# Announcements File
# ============================================================
#
# This file defines ALL server announcements.
#
# Announcements can be defined EITHER:
#   1) In this file (recommended for readability), OR
#   2) Inline in Announce.json
#
# If Announce.json contains:
#   "messages": "@file:announcements.txt"
# then THIS file is used.
#
# ----------------------------
# Announcement Types
# ----------------------------
#
# @Schedule
#   - Runs automatically at the configured interval
#   - Can ALSO be called using /broadcast <name>
#
# @Broadcast
#   - Does NOT run automatically
#   - Can ONLY be called using /broadcast <name>
#
# ----------------------------
# File Structure
# ----------------------------
#
# @Type          ‚Üê either @Schedule or @Broadcast, case-insensitive
#
# [Name]          ‚Üê broadcast name (case-insensitive, must be one word)
# Message line    ‚Üê one chat line per line
#
# Notes:
#   - Lines starting with # are comments
#   - Formatting, click events, hover text, gradients,
#     and placeholders all work here
#
# ----------------------------
# Inline JSON Example
# ----------------------------
#
# If you prefer doing all this inside the config.json, it can also be done like this.
#
# {
#   "announcements": {
#     "enabled": true,
#     "intervalSeconds": 1800,
#     "messages": {
#       "scheduled": {
#         "vote": [
#           "&cRemember to <click:/vote|&e/vote>",
#           "&aVoting helps grow the server!"
#         ]
#       },
#       "broadcast": {
#         "hub": [
#           "&6We need donations for the nether hub"
#         ]
#       }
#     }
#   }
# }
#
# ============================================================
